# 调用接口运行期执行特定逻辑的实现方法

[toc]

## 一、预习题

```
有一个接口，比如
public interface Demo {
    String doit(int i);
}
如果我们想要在运行期，调用这个接口的时候，去执行一定特定的逻辑，有多少种实现方式，提示例如动态代理就是一种。
```

思考：

1. 利用Java中的多态，通过继承抽象类或实现接口来实现；

2. 动态代理；

3. 利用Java的SPI；

   > https://blog.csdn.net/hefrankeleyn/article/details/123218329

4. AOP；

5. Java-agent;（带确认）
